# 📊 DashLogistics Data Dictionary

## Overview

This document describes all fields in the DashLogistics datasets. There are two primary datasets:

1. **enriched_data.csv** - Core shipping + fuel data (10 columns)
2. **logistics_analysis_enriched.csv** - Analysis-ready data with efficiency scores (12 columns)

---

## enriched_data.csv

**Location**: `data/final/enriched_data.csv`  
**Records**: 50 US states  
**Source**: ETL pipeline merge (shipping demographics + AAA fuel prices)  
**Update Frequency**: On demand (run `python main.py`)

### Field Definitions

| Field | Type | Description | Example | Source |
|-------|------|-------------|---------|--------|
| **rank** | integer | State population rank (1=most populous) | 5 | shipping_data.csv |
| **state** | string | US state postal code (uppercase) | CA | normalized from shipping_data.csv |
| **postal** | string | US state 2-letter code | CA | lookup via state_mapper.py |
| **population** | float | Total state population (2020 Census est.) | 38,802,500 | shipping_data.csv |
| **population_per_rank** | float | Computed metric: population / rank | 7,760,500 | ETL transformation |
| **regular** | float | Average regular gasoline price (USD/gal) | 5.368 | AAA web scraper |
| **mid_grade** | float | Average mid-grade gasoline price (USD/gal) | 5.589 | AAA web scraper |
| **premium** | float | Average premium gasoline price (USD/gal) | 5.774 | AAA web scraper |
| **diesel** | float | Average diesel fuel price (USD/gal) | 6.209 | AAA web scraper |
| **pipeline_run_id** | string | Unique execution tracking ID (UUID) | a14ebd7bb... | Generated at pipeline start |

### Data Quality

- **Missing Values**: None (all 50 states present)
- **Validation**: Pydantic schemas enforce data types + constraints
- **Outliers**: None detected (realistic fuel & population ranges)
- **Correlation**: Diesel-Regular prices = 0.83 (strong positive)

### Notes

- **State Normalization**: All state names converted to 2-letter postal codes via centralized mapper
- **Fuel Prices**: Scraped from AAA website (real-time data)
- **DC Excluded**: Washington D.C. filtered out (not a state)
- **52 → 50 transformation**: Original data had 52 records, 2 filtered during ETL

## logistics_analysis_enriched.csv

**Location**: `data/final/logistics_analysis_enriched.csv`  
**Records**: 50 US states (sorted by efficiency_score descending)  
**Source**: enriched_data.csv + analysis calculations  
**Generated By**: `notebooks/01_LOGISTICS_ANALYSIS_REPORT.ipynb` Cell 21

### Field Definitions

| Field | Type | Description | Example | Formula/Range |
|-------|------|-------------|---------|----------------|
| **state** | string | US state 2-letter code | CA | - |
| **postal** | string | Postal code (synonym for state) | CA | - |
| **region** | string | US geographic region | West | Northeast / South / Midwest / West |
| **population** | float | State population | 38,802,500 | From shipping_data |
| **regular** | float | Regular gas price (USD/gal) | 5.368 | From AAA scraper |
| **mid_grade** | float | Mid-grade gas price (USD/gal) | 5.589 | From AAA scraper |
| **premium** | float | Premium gas price (USD/gal) | 5.774 | From AAA scraper |
| **diesel** | float | Diesel price (USD/gal) | 6.209 | From AAA scraper |
| **fuel_burden_index** | float | Population-weighted fuel cost | 6.209 | (pop / max_pop) × diesel |
| **efficiency_score** | float | Efficiency metric (higher = better) | 62,493.96 | population / (diesel × 100) |
| **efficiency_percentile** | float | Efficiency rank (0-100) | 100.0 | Percentile rank of efficiency_score |
| **efficiency_tier** | string | Performance category | Top Tier (Highly Efficient) | Based on efficiency_percentile threshold |

### Efficiency Scoring Methodology

**Efficiency Score** measures a state's logistics value relative to fuel costs:

```
efficiency_score = population / (diesel_price × 100)
```

**Interpretation**:
- **Higher score** = More population served per unit of fuel cost = Better efficiency
- **Lower score** = More expensive operations per capita = Worse efficiency

**Efficiency Tiers** (based on efficiency_score percentile):

| Tier | Percentile Range | Count | Interpretation |
|------|------------------|-------|-----------------|
| **Top Tier** | ≥ 75th | ~13 states | Highly efficient, optimal for logistics hubs |
| **Mid Tier** | 50-75th | ~13 states | Average efficiency, acceptable operations |
| **Low Tier** | 25-50th | ~12 states | Below average, requires optimization |
| **Bottom Tier** | < 25th | ~12 states | Least efficient, apply premium surcharges |

### Regional Breakdown

**Northeast** (9 states): CT, ME, MA, NH, RI, VT, NJ, NY, PA
- Avg Diesel: $4.989/gal (highest)
- Efficiency: Below average
- Characteristic: Higher population density, premium fuel costs

**South** (16 states): DE, FL, GA, MD, NC, SC, VA, WV, AL, KY, MS, TN, AR, LA, OK, TX
- Avg Diesel: $4.729/gal
- Efficiency: Best performers (highest avg efficiency_score)
- Characteristic: Large states, mixed costs

**Midwest** (12 states): IL, IN, MI, OH, WI, IA, KS, MN, MO, NE, ND, SD
- Avg Diesel: $4.494/gal (lowest)
- Efficiency: Above average due to lower fuel costs
- Characteristic: Most competitive fuel pricing

**West** (13 states): AZ, CO, ID, MT, NV, NM, UT, WY, AK, CA, HI, OR, WA
- Avg Diesel: $4.937/gal
- Efficiency: Highly variable (CA low efficiency, smaller states higher)
- Characteristic: Geographic dispersion, varied terrain

### Sorting

Records are **sorted by efficiency_score descending**, so:
- Row 1 = Most efficient state
- Row 50 = Least efficient state

## Data Pipeline Lineage

```
shipping_data.csv (52 records)
       ↓
   [ETL Clean, Validate, Normalize]
       ↓
enriched_data.csv (50 records) + PostgreSQL
       ↓
   [Analysis in Jupyter]
       ↓
logistics_analysis_enriched.csv (50 records)
```

### Audit Trail

- **pipeline_run_id**: Unique identifier for each pipeline execution (UUID)
- All records in enriched_data.csv have the same **pipeline_run_id**
- Enables tracking data lineage: which records came from which run

## Database Tables

Two main tables in PostgreSQL (`shipping_db`):

### shipping_stats
Mirrors enriched_data.csv structure

```sql
SELECT * FROM shipping_stats WHERE state = 'CA';
-- rank | state | postal | population | population_per_rank | pipeline_run_id
```

### fuel_prices
Denormalized fuel price data with timestamp

```sql
SELECT * FROM fuel_prices WHERE state = 'CA';
-- state | regular | mid_grade | premium | diesel | scraped_at | pipeline_run_id
```

## Usage Examples

### Load and Explore

```python
import pandas as pd

# Load enriched analysis data
df = pd.read_csv('data/final/logistics_analysis_enriched.csv')

# Filter Top Tier states
top_tier = df[df['efficiency_tier'] == 'Top Tier (Highly Efficient)']
print(f"Top tier states: {top_tier['state'].tolist()}")

# Regional analysis
south_region = df[df['region'] == 'South']['diesel'].mean()
print(f"South avg diesel: ${south_region:.3f}")

# Find highest burden
high_burden = df.nlargest(5, 'fuel_burden_index')[['state', 'population', 'fuel_burden_index']]
```

### Query from PostgreSQL

```python
import sqlalchemy as sa

engine = sa.create_engine('postgresql://postgres:admin@localhost:5432/shipping_db')

# Top 10 efficient states
query = "SELECT state, efficiency_score FROM shipping_stats ORDER BY efficiency_score DESC LIMIT 10"
result = pd.read_sql(query, engine)
```

## Data Quality & Limitations

✅ **Strengths**:
- Complete coverage (all 50 states)
- Real-time fuel prices from authoritative source (AAA)
- Population data from 2020 Census (stable/reliable)
- Consistent state normalization across all fields

⚠️ **Limitations**:
- Fuel prices are point-in-time snapshots (not time series)
- Weather data optional (API key dependent)
- Population data is fixed (2020 Census, not updated annually)
- Limited to 50 states + DC (no territories)
